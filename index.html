Ragial Notifier

Purpose

A very popular MMORPG game, Ragnarok Online, developed by Gravity Inc. has a companion site called ragial.com which tracks all of the items vended by players ingame. However, it becomes tedious to regularly refresh the ragial.com site, and thus this application was developed to help players track items that they wish to buy or sell.

This is an Android Application designed to parse ragial.com using the Ragial Searcher java library also developed by me. Its primary task is to parse ragial.com for items that the user wishes to track and to notify at certain intervels if item/s are on sale.

Link : https://play.google.com/store/apps/details?id=com.somshubra.ragialnotifier
